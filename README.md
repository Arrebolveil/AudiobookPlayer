# 悦听有声书 (Audiobook Player)

一款基于 Tauri v2 + Vue 3 开发的本地桌面端有声书播放器。设计极简，功能专注，旨在提供无缝的本地有声读物收听体验。

## ✨ 核心特性

- **多书籍 / 多专辑管理**：支持一键导入含有多个子文件夹的书库目录，自动将不同子文件夹识别为独立的书籍。
- **支持拖拽导入**：支持将书库文件夹直接拖拽进软件界面进行快速挂载。
- **独立进度记忆**：每本书拥有完全独立的播放进度。在不同书籍间切换时，能自动保存并恢复各自的收听位置（精确到秒）。
- **高精度进度条**：自研进度条组件，支持精准的鼠标拖拽定位，并提供快捷的 `-15秒` 和 `+15秒` 跳转。
- **自然语义排序**：智能识别音频文件名（如“第2集.mp3”和“第10集.mp3”），确保按正常人类逻辑顺序播放。
- **丰富的定制播放体验**：
  - 内置 0.75x 到 2.0x 六档倍速控制。
  - 支持应用内快捷键（空格键控制播放/暂停，左右方向键微调进度）。
  - 支持全局快捷键（即使软件在后台运行，也可以通过设定的组合键进行控制）。
- **托盘常驻模式**：支持关闭窗口时最小化到系统右下角托盘，提供右键菜单控制播放、切换歌曲及安全退出。

## 🛠️ 技术栈

- **前端**：Vue 3 + TypeScript + Tailwind CSS
- **后端**：Rust + Tauri v2 框架
- **存储**：轻量级本地 localStorage 数据持久化

## 🚀 本地开发与运行

确保您的电脑已安装 [Node.js](https://nodejs.org/) 和 [Rust 运行环境](https://www.rust-lang.org/tools/install)。

```bash
# 1. 克隆项目
git clone https://github.com/您的用户名/您的仓库名.git
cd AudiobookPlayer

# 2. 安装前端依赖
npm install

# 3. 启动开发环境
npm run tauri dev

# 4. 构建生产版本安装包
npm run tauri build
```
